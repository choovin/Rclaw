## 七、数据结构

### 7.1 Employee 类型

```typescript
interface Employee {
  id: string;
  name: string;           // 英文名
  nameZh: string;         // 中文名
  description: string;    // 介绍
  color: string;          // 主题色
  emoji: string;          // emoji
  vibe: string;           // 风格描述
  department: Department; // 部门

  // 增强字段
  soulContent?: string;
  agentsContent?: string;
  identityContent?: string;
}
```

### 7.2 EmployeeWithStatus 类型

```typescript
interface EmployeeWithStatus extends Employee {
  isAdded: boolean;
  agentId?: string;
  addedAt?: number;
}
```

### 7.3 Department 枚举

```typescript
type Department =
  | 'engineering'
  | 'design'
  | 'marketing'
  | 'sales'
  | 'product'
  | 'project-management'

  | 'strategy'
```
