# Rclaw v0.2.5 版本更新 - 知识文档

**创建时间**：2026-03-19
**最后更新**：2026-03-19
**版本**：v1.0

---

## 一、版本概述

本次更新包含以下主要变更：
1. 品牌重命名：ClawX → Rclaw
2. 左下角版本号显示
3. 合并远端修复：Linux 中文 IME 问题
4. 员工市场性能优化

---

## 二、变更详情

### 2.1 品牌重命名

| 文件 | 变更 |
|------|------|
| `package.json` | name: rclaw, version: 0.2.5 |
| `electron-builder.yml` | productName: Rclaw, appId: app.rclaw.desktop |
| `electron/main/index.ts` | 日志文本：ClawX → Rclaw |
| `electron/preload/index.ts` | versions.app: '0.2.5' |

### 2.2 版本号显示

**新增文件**：
- `src/components/layout/VersionDisplay.tsx`

**功能**：
- 左下角显示 Rclaw 版本号
- Gateway 连接后显示 OpenClaw 版本号

**显示格式**：`Rclaw v0.2.5 | OpenClaw v{x.x.x}`

**相关代码**：
```tsx
// MainLayout.tsx 引入组件
import { VersionDisplay } from './VersionDisplay';

// 在布局底部添加
<VersionDisplay />
```

### 2.3 Linux IME 修复

**合并自远端**：`1b527d2` - fix(linux): Can't change Chinese IMEs on Debian (#582)

| 文件 | 变更 |
|------|------|
| `electron/main/index.ts` | Linux 使用原生窗口边框 |
| `src/components/layout/TitleBar.tsx` | Linux 平台返回 null |

### 2.4 员工市场优化

**文件**：`src/pages/Agents/Marketplace.tsx`

**变更**：
- 从 fetch 改为直接 import JSON
- 减少网络请求，提升加载速度
