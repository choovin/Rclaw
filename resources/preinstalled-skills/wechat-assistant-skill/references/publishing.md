# 公众号文章发布规范

## 核心规则

1. **只创建草稿到草稿箱**，发布需人工手动操作
2. 草稿箱地址：https://mp.weixin.qq.com → 内容与互动 → 草稿

## 配图要求

- 每篇文章至少 3 张文中配图
- 配图比例：16:9 横屏
- 配图 prompt 后缀："16:9横屏构图"
- AI 生图后必须检查文字是否有乱码
- 图片必须与文章内容相关

## 内容规范

- 标题要吸引人但不标题党
- 摘要简明扼要，突出核心价值
- 结构清晰，分段合理
- 适当使用表情符号增加亲和力
- 不涉及敏感政治话题，不传播虚假信息

## HTML 格式注意

- Markdown 表格无法被标准转换器转为 HTML `<table>`，需手动处理
- 转换后删除表格分割线 `|------|` 行
- 表头配色：深色背景(#2c3e50) + 白色字
- 清理多余的 `<hr style="margin: 24px 8px..." />` 分隔符

## 图片插入注意

- 用唯一锚点文字（段落最后一句话）定位插入位置
- 不要依赖图片 URL 去重，可能错误替换
- 插入后检查是否有嵌套 `<p>` 标签
